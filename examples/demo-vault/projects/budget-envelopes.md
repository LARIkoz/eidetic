# Budget Envelopes

A zero-based budgeting app that splits income into virtual envelopes stored in a single SQLite file. Each envelope has a cap, a spend log, and a rollover rule. The UI is a terminal dashboard so it runs on any machine without a browser. Import from CSV bank statements is the current work-in-progress.
