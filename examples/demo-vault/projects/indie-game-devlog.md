# Indie Game Devlog

A structured devlog system for solo game developers that stores each entry as a Markdown note with frontmatter tags for milestone, feature area, and build number. Entries are indexed with full-text search so a developer can quickly find when a mechanic was introduced or a bug first appeared. An export script produces a static HTML timeline for publishing.
