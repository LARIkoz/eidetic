# Plant Care Reminder

A notification-based app that schedules watering, fertilising, and repotting reminders per plant species. The species database is a bundled SQLite file seeded from a curated open dataset. Reminders adapt to the local climate zone by pulling a daily temperature forecast and adjusting intervals accordingly.
