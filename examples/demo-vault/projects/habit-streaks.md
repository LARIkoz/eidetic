# Habit Streaks

A minimalist streak counter that tracks daily habits in a flat Markdown file per habit. Each day is a line with a date and a check mark or skip reason. A companion script generates a heatmap calendar as an SVG for embedding in personal dashboards. The design goal is zero dependencies beyond a standard Python install.
