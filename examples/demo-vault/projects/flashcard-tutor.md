# Flashcard Tutor

A spaced-repetition flashcard app that uses vector similarity to suggest related cards when a learner struggles with a topic. Cards are stored as JSON Lines, one object per line, which makes diffing and version control straightforward. The scheduling algorithm is a simplified variant of SM-2 with an added fatigue modifier.
