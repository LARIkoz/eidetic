# Trail Run Tracker

A lightweight GPS logger for trail runners that records routes as GeoJSON and syncs them to a personal SQLite database. Elevation gain and split paces are calculated on-device to avoid sending location data to third-party servers. The next feature is a weekly summary widget that ranks routes by effort score.
