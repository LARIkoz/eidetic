# Recipe Keeper

A local-first recipe manager that stores dishes as plain Markdown files with YAML frontmatter for ingredients and cook times. Full-text search is powered by an embedded SQLite database so the app works offline. The current milestone is adding image thumbnails generated from the first photo in each recipe note.
