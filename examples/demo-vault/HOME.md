## What this demo shows

Eidetic is an AI memory system for Claude Code. It exports your accumulated session memory into an Obsidian-compatible vault — searchable, cross-linked, and human-readable.

This demo is a real export, filtered down to ~55 of the most representative notes and anonymized. Real exports typically contain 200-400 notes after a few months of use.

### Anonymization

This is a demo derived from real data. All personal identifiers have been replaced with generic placeholders:

- "User" and "partner" instead of personal names
- "my-project" / "extractor" / "medical" instead of real project slugs

The technical content (SQLite rules, API patterns, model selection logic) is unchanged — that's the value.
