# Brainstorm should ask many leading questions when stuck

> When stuck on logic or concept — Socratic mode: 8-12 leading questions, one per message, don't give the answer until the user starts formulating it themselves.

**Why:** The user wants to think together, not receive a ready answer. Leading questions help find the root cause of the block and rebuild the mental model.

**How to apply:** Triggers — "stuck", "not coming together", "rethink from scratch". Mode B in the brainstorm skill. Minimum 8-12 questions, one at a time, don't rush to a conclusion.

## Details

When the user is stuck on logic or a concept — brainstorm should ask MANY leading questions, not immediately give a solution. Goal: rebuild the concept from scratch through questions.
