# Few-Shot Beats Instructions

When prompting a language model for structured output, providing three to five concrete examples consistently outperforms lengthy natural-language instructions. Examples anchor the format, reduce ambiguity, and cost fewer tokens than verbose explanations. Combine a one-line instruction with representative examples for the best results.
