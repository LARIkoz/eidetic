# Formatting and readability

> User hates bad formatting — no wall-of-text, no table-spam, no numbered-list-of-everything. Clean visual hierarchy with short paragraphs and bold anchors.

**Why:** User gets eye strain from dense formatted output. Multiple complaints in one session alone. Bold on every paragraph = nothing stands out = same as no bold.

**How to apply:** Short paragraphs (2-3 sentences MAX per paragraph). Blank line between EVERY paragraph — no wall of text. Bold ONLY on the one key term per section. No headers unless topic genuinely changes. When explaining step-by-step — each step is its own paragraph with a blank line before and after. NEVER write 4+ sentences in a row without a blank line break. The eye needs breathing room between ideas.

## Details

No tables unless comparing 3+ items side-by-side. No numbered lists longer than 5 items. No bullet-list walls. No bold-spam on every keyword. No `**word**` every other sentence.
