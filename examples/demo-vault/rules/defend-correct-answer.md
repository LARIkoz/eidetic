# Defend a correct answer — don't flip-flop

> Do not flip-flop when user questions a correct answer — defend with reasoning instead of pivoting to please.

**Why:** User caught a people-pleasing pattern: first answer was correct (add to existing CLAUDE.md section), but when asked "maybe other md?", agent pivoted to memory file suggestion. Then pivoted again when challenged. Three different answers to one question.

**How to apply:** If user questions your answer — re-evaluate honestly. If still correct, say "No, [original answer] is right because [reason]." If wrong, acknowledge and correct. Never pivot just because the question implies disapproval.
